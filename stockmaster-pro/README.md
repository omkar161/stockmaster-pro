# StockMaster Pro - Product Inventory Management System

A full-stack inventory management system built with **React + Vite (Frontend)** and **Node.js + Express + SQLite (Backend)**.

## Live Links
- Frontend: https://stockmaster-pro.vercel.app
- Backend: https://stockmaster-pro.up.railway.app
- Repo: https://github.com/omkarpawar05/stockmaster-pro

## Features
- Add/Edit/Delete products with inline editing
- CSV Import/Export (with duplicate detection)
- Real-time search & category filtering
- Inventory change history panel
- Stock status color coding
- Fully responsive + Dark mode
- Premium glassmorphism UI
